# DicomVision v1.6.0

相比 v1.5.0，本版本主要更新：

- Web Demo 支持服务器端示例影像加载，同时保留上传 DICOM 文件和上传 DICOM 文件夹入口。
- Web Demo 已上传 P113 4D CT 示例数据，线上可直接加载 10 个 phase 的示例影像。
- 公开部署安全性增强：隐藏 `/docs`、`/openapi.json`、`/redoc`，上传目录支持自动清理。
- Web 部署链路优化，适配京东云等国内服务器环境，降低部署包体积并提升 Docker 构建稳定性。
- PACS Browser 的下拉框、日期选择、选中态、列表滚动和已下载序列提示继续统一和完善。
- PACS 默认配置保留 Orthanc、dcm4chee，其他 PACS 类型仅作为创建选项；DIMSE 默认端口按协议类型选择。
- DICOM SR 与 DICOM GSPS 导出和重新导入支持继续完善，GSPS 文案改为更易理解的标注/测量叠加。
- 3D/MPR 视图隐藏不适用的 DICOM SR/GSPS 导出入口，3D 参数面板增加关闭入口。
- 移除 Windows 资源管理器 DICOM 预览扩展相关代码，避免资源管理器卡顿。
- 桌面端打包支持将后端一起打入安装包，并补充 macOS 打包脚本基础支持。
- Web/Electron 上传入口、图标、品牌区和多主题样式进一步统一；补齐多处中英文界面文案。
- README 改为中文优先，并保留英文文档入口。

Assets:

- `DicomVision-1.6.0-Setup.exe`: Windows 安装包，已包含桌面端前端和后端运行包。
- `DicomVision-1.6.0-Setup.exe.blockmap`: 自动更新差分信息。
- `latest.yml`: Electron 更新元数据。
