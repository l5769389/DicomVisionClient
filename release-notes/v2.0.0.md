# DicomVision v2.0.0

Compared with v1.6.0, this release focuses on MPR/4D viewing, interaction performance, workspace isolation, and a cleaner viewer UI.

## Highlights

- Added Double Oblique support for MPR and 4D MPR, including orthogonal lock switching and reset behavior.
- Improved 4D MPR phase handling so phase state, MPR pose, and interaction mode stay consistent across phases.
- Added anonymous tab workspace isolation for the web version, reducing cross-tab and multi-user state interference.
- Improved Stack and MPR interaction responsiveness for window/level, pan, zoom, crosshair drag, and oblique rotation.
- Added MPR MIP slab projection support with 0-100 mm thickness controls, 10 mm defaults, oblique support, and slab boundary guide lines.
- Added viewer display controls for corner information and scale bar visibility.
- Refined the toolbar, sidebar, tab layout, GitHub entry, and desktop/web visual consistency.
- Added Windows startup scripts for web and desktop development workflows.

## Assets

- `DicomVision-2.0.0-Setup.exe`: Windows installer with the desktop frontend and bundled backend runtime.
- `DicomVision-2.0.0-Setup.exe.blockmap`: Differential update metadata.
- `latest.yml`: Electron update metadata.
