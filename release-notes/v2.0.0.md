# DicomVision v2.0.0

相比 v1.6.0，本版本重点增强 MPR/4D 浏览、交互性能、多用户隔离和界面体验。

## 主要更新

- 新增 MPR 与 4D MPR 的 Double Oblique 支持，可在正交锁定和 Double Oblique 之间切换，并支持重置恢复。
- 优化 4D MPR phase 行为，phase 切换时 MPR 姿态、交互模式和视图状态保持一致。
- Web 版本新增匿名 tab workspace 隔离，减少多 tab 或多用户同时访问时的影像、视图和任务状态串扰。
- 提升 Stack 与 MPR 常用交互体验，包括调窗、平移、缩放、十字线拖动和 oblique 旋转。
- 新增 MPR MIP 厚层预览能力，支持 0-100 mm 层厚、默认 10 mm、oblique 视图和厚度范围虚线提示。
- 新增视图显示选项，可控制四角信息和比例尺的显示。
- 优化工具栏、侧边栏、tab 布局、GitHub 入口，以及桌面端和 Web 端的视觉一致性。
- 新增 Windows 脚本，便于启动 Web 前后端和桌面端开发环境。

## 安装包

- `DicomVision-2.0.0-Setup.exe`：Windows 安装包，包含桌面端前端和内置后端运行包。
- `DicomVision-2.0.0-Setup.exe.blockmap`：自动更新差分信息。
- `latest.yml`：Electron 自动更新元数据。
