# DicomVision v3.1.0

v3.1.0 focuses on 3D rendering quality, mobile 3D usability, adaptive presets, remove-bed/clipping workflows, local demo data, and LAN-friendly web development.

## Assets

- `DicomVision-3.1.0-Setup.exe` - Windows desktop installer with the backend bundled.
- `DicomVision-web-v3.1.0.zip` - static web frontend build.
- `DicomVisionServer-v3.1.0-win-x64.zip` - standalone Windows backend bundle.
- `DicomVision-3.1.0-Setup.dmg` - macOS desktop installer image.
- `DicomVision-3.1.0-Setup.zip` - macOS desktop ZIP package.
- `SHA256SUMS-windows.txt` / `SHA256SUMS-macos.txt` - release asset checksums.

## Highlights

- Improved 3D VR/Surface preview and final-frame consistency.
- Added direct model drag rotation, interactionId stale-frame suppression, and better mobile viewport fitting.
- Expanded 3D preset catalog with General, CT, CTA, MR, and CBCT groups.
- Made AAA and other presets adaptive using CT HU anchors plus foreground percentiles.
- Added Surface-specific parameter handling and adaptive iso-surface defaults.
- Added remove-bed rendering, freeform 3D clipping, clipping progress feedback, and mobile 3D tool support.
- Improved local web development so LAN clients connect to the backend through the current page host.
- Disabled stale PWA service worker behavior in development to avoid module MIME errors.
- Updated local macOS demo loading to prefer the configured 320-slice test dataset while production keeps the bundled sample.

## Validation

- Frontend type checking and targeted Vitest coverage for runtime backend resolution and PWA cache cleanup.
- Backend tests for sample loading, 3D rotation, volume preset adaptation, surface rendering, socket state, and clip/removeBed behavior.
