# DicomVision v3.1.1

DicomVision v3.1.1 is a maintenance release focused on stable cross-platform viewing, 2D/MPR interaction quality, 3D workflow consistency, and release packaging.

## Highlights

- Fixed-width, backend-authoritative X/Y and pixel-value readout in the bottom-right corner of 2D, Compare, Layout, and MPR viewports. The display now uses `X: 210 Y:  99    115 HU`, preserves the last valid value, and avoids hover-induced layout jitter.
- Improved 2D and MPR server-side pan/zoom behavior, pointer-up state handling, hover sampling, trackpad pinch zoom, touch gestures, and coordinate stability.
- Preserved off-screen measurement and annotation geometry during pan and zoom instead of clamping points to the viewport.
- Improved 3D VR/Surface interaction, orientation selection, mobile controls, layout view rendering, viewport overlays, surface presets, remove-bed processing, and freeform clipping feedback.
- Added regression coverage for viewer lifecycle, tabs, layouts, rendering presets, gestures, hover values, surface rendering, socket state, and 3D workflows.
