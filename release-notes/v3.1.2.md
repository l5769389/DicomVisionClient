# DicomVision v3.1.2

DicomVision v3.1.2 improves 3D rendering observability and throughput, refines the desktop workspace, and expands regression coverage for frequently used viewer interactions.

## Highlights

- Added startup VTK/OpenGL diagnostics and repeatable 3D rendering benchmarks, including separate timings for VTK rendering, GPU readback, WebP encoding, GPU-process IPC, and Socket.IO delivery.
- Added an optional dedicated GPU rendering process on macOS to isolate VTK state from API and Socket.IO workers and improve renderer lifecycle stability.
- Preserved integral 16-bit volume data where possible instead of always expanding DICOM volumes to `float32`, reducing memory traffic and rendering-process transfer cost.
- Optimized 3D WebP output for interactive and final frames while keeping high-quality settled images and backend-authoritative render revisions.
- Refined the desktop tool and result docks with contextual help popovers, consistent reset actions, improved paging controls, pseudocolor reset, and continuous left/right panel resizing.
- Added and expanded tests for dock resizing, contextual help, toolbar actions, MTF results, 4D views, GPU rendering-process behavior, volume data types, and MPR rendering.
