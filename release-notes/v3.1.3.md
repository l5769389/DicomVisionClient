# DicomVision v3.1.3

DicomVision v3.1.3 consolidates the remote 3D rendering workflow, strengthens upload safety, and finishes a broad desktop, mobile, and theme-consistency pass.

## Highlights

- Added startup-configured 3D transport: use WebRTC for low-latency interactive VR/Surface previews, while keeping a lossless WebP settled frame for diagnostic clarity.
- Improved 3D frame handoff, stale-frame suppression, final-frame quality, GPU-process diagnostics, VTK/WebP/WebRTC benchmarks, and Surface mesh reuse.
- Refined the 3D workspace with stable VR/Surface switching, orientation controls, remove-bed state, freeform clipping, progress feedback, viewport fitting, and mobile controls.
- Unified desktop and mobile tool behavior, selection states, icon sizing, dock help, tabs, settings navigation, theme tokens, and bilingual copy across the industrial dark, cool blue, and clinical light themes.
- Improved 2D/MPR pan, zoom, pinch and trackpad gestures, hover readout persistence, HU display, overlays, MIP/segmentation panels, paging controls, and layout behavior.
- Added safe compressed-study import for ZIP, 7z, and RAR. Imports are extracted into temporary sessions and protected by path, entry-count, unpacked-size, and compression-ratio checks.
- Expanded regression coverage for rendering transport, tabs, tool state, mobile interactions, MPR, uploads, archive import, and server view lifecycle.

- Windows standalone Server bundle: `DicomVisionServer-v3.1.3-win-x64.zip`
